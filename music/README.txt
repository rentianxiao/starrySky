背景音乐说明
================

1. 找一首你喜欢的抒情纯音乐（钢琴、吉他等均可）
2. 下载为 MP3 格式
3. 重命名为：bg.mp3
4. 放在本文件夹（与 index.html 同级路径：love-letter/music/bg.mp3）

版权提示：请使用你有权使用的音乐（自己制作、已获授权、或明确标注可商用的免费素材）。

推荐免费素材站（需自行下载后重命名）：
- 爱给网 aigei.com（搜索「浪漫钢琴」「抒情」）
- 淘声网 tosound.com

上传 GitHub 时记得把 bg.mp3 一起 git add，否则网页没有声音。
